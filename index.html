Conectar ao Supabase

Configure uma vez e seus lançamentos e metas ficam sincronizados em qualquer dispositivo automaticamente.

Como criar seu projeto gratuito:
1. Acesse supabase.com → New Project
2. Em Project Settings → API copie a Project URL e a chave anon public
3. Em SQL Editor rode o script abaixo para criar as tabelas:

CREATE TABLE financas_meses ( id text PRIMARY KEY, data jsonb NOT NULL, updated_at timestamptz DEFAULT now() ); CREATE TABLE financas_metas ( id text PRIMARY KEY DEFAULT 'metas', data jsonb NOT NULL, updated_at timestamptz DEFAULT now() );
